Project Portfolio — Craig Stueber
Chat Assistant

Description:
A safety-focused conversational assistant designed for my personal website, built to explore structured intent handling, controlled model outputs, and responsible LLM feature integration.

Business Impact:
Demonstrates AI-assisted UX that prioritizes user trust, predictable behavior, and guardrail-driven interactions aligned with modern AI safety principles.

Highlights:

Built an intent-routing system that categorizes queries into safe, deterministic workflows before calling any model

Designed guardrails and fallback responses to avoid unsafe or ambiguous model behavior

Implemented reasoning-chain compression to ensure concise, stable answers without exposing system prompts

Added structured memory (context windows, state machines) for multi-step interactions without unintended persistence

Extending the system with constitutional-style rules for tone, safety, and clarity

Technologies: React, Next.js, TypeScript, Python, LLM APIs, Prompt Engineering
